### ✨ Features
- 🔐 **Secure Authentication:** JWT-based user and admin authentication.
- 💳 **Payment Integration:** Stripe & Razorpay support.
- ☁️ **Cloud Storage:** Cloudinary for image management.
- 📦 **Order Management:** Admin panel for tracking and status updates.
- 🐳 **DevOps Ready:** Fully containerized with Docker & AWS EC2 deployment.
